= 前后端分离的接口设计

[NOTE]
====
数据通信方式和 get(获取) 和 post(提交);

数据交换格式为 json;

通信端口：26000

====

## 登录接口

[width="100%"]
|====================
| URL       2+|/device/login
|  http method 2+|  post
3+|data
| userId | 用户名|  "admin"
|  password| 密码|  "admin"
3+|数据返回
| ret | 结果 |0:成功 -1:失败
| msg | 返回消息（String）|
| data | 数据 |
|====================

| ipSync     | ip取得方法  | 0:固定 ip;1：DHCP
| ip         | ip          |
| port       | port        |
| gateway    | 网关        |
| subnet     | 子网        |
| room       | 房间编号    |
| command    | 命令        |
| serverIp   | 服务器IP？  |
| serverPort | 服务器 Port |
| timeout    | 超时        |
| duration   | 开门延迟    |
| cardEncrypt  | 卡号加密    | 0;1
| cardType   | 卡号加密(String)    |"有効"/"無効"
| decryptKey | 解密密钥    |
| decryptIv  | 解密IV      |
| userId   | 用户名    |
|====================
